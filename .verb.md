## CLI

In the command line:

```bash
deps
```

### Commands

 - `-r` | `--report`: save a report to disk. By default `report.json` is saved. To change the path do: `-r foo.json`
 - `-e` | `--exclude`: comma-separated list of files or directories to exclude. Example: `deps -e test,lib`
